Attributes

data  [RW]  This is the data for the graph. It should be an array where every item is either a number or an array of numbers.

Examples: For a simple bar graph: data=[1,2,3] For a grouped bar graph: data=[[1,10],[2,20],[3,30]]

labels  [RW]  An array of labels to use on the y axis. Make sure you have the right number of labels. The size of this array should = the size of the data array.

Public Class methods

new(options = {})

Returns a new graph creator with some default options and other options that you pass.

The options has includes: :height - String to use as height parameter on the svg tag. Default is ‘100%’. :width - String to use as width parameter on the svg tag. Default is ‘100%’. :indent - Indent option to the XmlMarkup object. Defaults to 2. :padding - Number of svg units in between two bars. :viewbox - Hash of :height and :width keys to use for the viewbox parameter of the svg tag. Defaults to { :height => 600, :width => 800 }. :margin - Hash of margins to use for graph (in svg units). Defaults to { :top => 50, :left => 50, :right => 50, :bottom => 100 }.

render() {|xml, @options if block_given?| ...}

This returns a string of the graph as an svg. You can pass in a block in order to add your own items to the graph. Your block is passed the XmlMarkup object to use as well as the options hash in case you need to use some of that data.
